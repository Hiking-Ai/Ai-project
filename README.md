
# 맞춤형 산행 안전 탐방로 추천 서비스

## 1. 프로젝트 개요

### 선정 배경
- 봄·가을철 산을 이용하는 인구 증가  
- 급변하는 기상 상황으로 인한 안전사고 위험 증대

### 목표
- **등산객 맞춤 탐방로 추천**  
- 기상정보 및 산악 특보를 반영한 안전 탐방 지원  
- 숙박·휴양림·체험시설 등 주변 인프라 정보 제공  
- 대중교통(버스·터미널) 연동을 통한 접근성 강화

### 기대효과
- 사용자의 산행 안전사고 예방  
- 주변 지역 관광·숙박업 등 경제 활성화

## 2. 기술 스택 & 사용 툴

- **프론트엔드**: React  
- **백엔드**: FastAPI  
- **데이터베이스**: MySQL  
- 프로젝트 관리: Notion  
- 버전 관리: GitHub
